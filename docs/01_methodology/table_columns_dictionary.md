# Словарь столбцов табличных отчетов

Дата обновления: `2026-05-19`.

Документ связывает технические названия столбцов в CSV/XLSX-выгрузках с русскими наименованиями для отчетов, презентаций и dashboard.

## Общие отчетные поля

| Техническое имя | Русское название | Комментарий |
|---|---|---|
| `report_period_label` | Отчетный период | Метка периода: месяц, квартал или год. |
| `report_year` | Отчетный год | Год, к которому относится отчетный период. |
| `report_period_type` | Тип отчетного периода | `month`, `quarter` или `year`. |
| `report_period_start` | Начало отчетного периода | Первая дата периода. |
| `report_period_end` | Конец отчетного периода | Последняя дата периода. |
| `is_target_period` | Целевой отчетный период | Признак основного периода без ретроспективного сдвига. |
| `auction_count` | Количество размещений | Количество строк/размещений в группе расчета. |
| `data_quality_flag` | Флаг качества данных | Методическое ограничение или `ok`. |

## Таблица доходности по видам ОФЗ

| Техническое имя | Русское название | Комментарий |
|---|---|---|
| `ofz_type` | Вид ОФЗ | Тип бумаги из исходных данных. |
| `yield_min` | Минимальная доходность | Минимальное значение доходности в группе. |
| `yield_weighted_avg` | Средневзвешенная доходность | Взвешивается по объему размещения, если он доступен. |
| `yield_max` | Максимальная доходность | Максимальное значение доходности в группе. |
| `yield_min_yoy_change` | Годовое изменение минимальной доходности | Разница к сопоставимому периоду прошлого года. |
| `yield_weighted_avg_yoy_change` | Годовое изменение средневзвешенной доходности | Разница к сопоставимому периоду прошлого года. |
| `yield_max_yoy_change` | Годовое изменение максимальной доходности | Разница к сопоставимому периоду прошлого года. |

## Таблица спроса и предложения

| Техническое имя | Русское название | Комментарий |
|---|---|---|
| `total_demand` | Совокупный спрос | Сумма поля `Совокупный объем спроса по номиналу`; учитываются аукционы и ДРПА при наличии значения. |
| `total_supply` | Совокупное предложение | Сумма поля `Объем предложения`. |
| `total_demand_yoy_change` | Годовое изменение совокупного спроса | Разница к сопоставимому периоду прошлого года. |
| `total_supply_yoy_change` | Годовое изменение совокупного предложения | Разница к сопоставимому периоду прошлого года. |
| `demand_supply_ratio` | Отношение спроса к предложению | `total_demand / total_supply`. |
| `demand_supply_ratio_yoy_change` | Годовое изменение отношения спроса к предложению | Разница к сопоставимому периоду прошлого года. |

## Таблица объемов размещения по срокам обращения

| Техническое имя | Русское название | Комментарий |
|---|---|---|
| `maturity_bucket` | Категория срока обращения | `short_term`, `medium_term`, `long_term`, `requires_review`. |
| `maturity_bucket_label` | Название категории срока обращения | Русская человекочитаемая категория. |
| `placement_volume` | Объем размещения | Сумма объема размещения по категории срока. |
| `placement_volume_yoy_change` | Годовое изменение объема размещения | Разница к сопоставимому периоду прошлого года. |
| `placement_volume_share` | Доля в общем объеме размещения | Доля категории срока в суммарном размещении периода. |
| `placement_volume_share_yoy_change` | Годовое изменение доли размещения | Разница доли к сопоставимому периоду прошлого года. |

## Ratio-показатели

| Техническое имя | Русское название | Комментарий |
|---|---|---|
| `bid_to_cover_ratio` | Покрытие предложения спросом | `demand_volume / supply_volume`; настоящий bid-to-cover. |
| `demand_to_placement_ratio` | Отношение спроса к размещению | `demand_volume / placement_volume`; не называть bid-to-cover. |
| `demand_satisfaction_ratio` | Коэффициент удовлетворения спроса | Исходное поле, если доступно; иначе `placement_volume / demand_volume`. |
| `ratio_basis` | База расчета ratio-показателей | Описание примененной формулы и ограничений знаменателя. |

## Базовые поля данных

| Техническое имя | Русское название | Комментарий |
|---|---|---|
| `auction_date` | Дата размещения | Основная дата аукциона/размещения. |
| `format` | Формат размещения | Например, `Аукцион` или `ДРПА`. |
| `format_assumption_flag` | Флаг определения формата | Показывает, был формат явным или восстановленным по правилу. |
| `issue_code` | Код выпуска | Идентификатор выпуска ОФЗ. |
| `security_type` | Тип бумаги | Исходное наименование типа бумаги. |
| `maturity_date` | Дата погашения | Дата погашения выпуска. |
| `days_to_maturity` | Дней до погашения | База для классификации срока обращения в таблицах. |
| `maturity_years` | Срок обращения в годах | Расчетный срок обращения. |
| `offer_amount_mln_rub` | Объем предложения, млн руб. | Исходное поле предложения. |
| `supply_volume` | Предложение | Нормализованный alias для `offer_amount_mln_rub`. |
| `demand_amount_mln_rub` | Совокупный объем спроса по номиналу, млн руб. | Исходное поле спроса. |
| `demand_volume` | Спрос | Нормализованный alias для `demand_amount_mln_rub`. |
| `placement_amount_mln_rub` | Объем размещения, млн руб. | Исходное поле размещения. |
| `placement_volume` | Объем размещения | Нормализованный alias для `placement_amount_mln_rub`. |
| `weighted_avg_yield_pct` | Средневзвешенная доходность, % | Исходное поле доходности. |
| `weighted_avg_yield` | Средневзвешенная доходность | Нормализованный показатель доходности. |
| `yield` | Доходность | Основной alias доходности для таблиц и графиков. |
| `cutoff_yield_pct` | Доходность по цене отсечения, % | Исходное поле доходности отсечения. |
| `proceeds_mln_rub` | Объем выручки, млн руб. | Исходное поле выручки. |
| `failed_or_no_deal` | Несостоявшееся или нулевое размещение | Признак строк, требующих осторожной интерпретации. |
