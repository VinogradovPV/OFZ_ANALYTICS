# Аналитическая архитектура проекта ОФЗ

Дата формирования: `2026-05-18`.

Архитектура поддерживает месячные, квартальные и годовые отчеты через параметры `report-date`, `retrospective-years` и `period-type`. Все аналитические блоки должны работать от единого параметризованного источника:

```text
data/processed/ofz_auctions_report_scope.csv
```

Обязательные фильтры и разрезы: `report_period_label`, `report_year`, `report_period_type`, `format`, `ofz_type`, `maturity_bucket`.

## 1. Общая аналитика аукционов ОФЗ

| Элемент | Содержание |
|---|---|
| Цель | Дать целостную картину первичных размещений ОФЗ за выбранный период и ретроспективу. |
| Управленческие вопросы | Как изменился объем размещений? Насколько устойчив календарь аукционов? Какие периоды дают структурный разрыв? |
| KPI | Количество аукционов, объем предложения, объем размещения, объем спроса, доля успешных размещений, `auction_efficiency_score`. |
| Визуализации | Динамика объемов, waterfall изменения размещения, heatmap активности, scatter спроса и доходности. |
| Табличные outputs | Сводная таблица по периодам; таблица спроса и предложения; таблица размещения по срокам. |
| Потенциальные инсайты | Рост/сжатие первичного рынка, изменение интенсивности заимствований, концентрация размещений в отдельных периодах. |
| Риск-сигналы | Падение размещения при высоком предложении, рост неразмещенного объема, снижение числа успешных аукционов. |

## 2. Аналитика доходности ОФЗ-ПД и других видов ОФЗ

| Элемент | Содержание |
|---|---|
| Цель | Оценить стоимость заимствований по ОФЗ-ПД как ключевому сегменту, а также по ОФЗ-ИН и другим видам ОФЗ при наличии надежных данных доходности. |
| Управленческие вопросы | Насколько изменилась доходность? Какие виды ОФЗ формируют давление на стоимость долга? Есть ли различия между ОФЗ-ПД, ОФЗ-ИН и другими инструментами? |
| KPI | `yield_min`, `yield_weighted_avg`, `yield_max`, `yield_yoy_change`, `yield_pressure_indicator`. |
| Визуализации | Линия доходности по периодам, boxplot доходностей по видам ОФЗ, heatmap доходности, scatter доходность/объем. |
| Табличные outputs | `ofz_yield_by_type_<period_type>_<report_date>_retrospective_<N>.csv/.xlsx`. |
| Потенциальные инсайты | Рост стоимости заимствований, расхождение доходности между типами ОФЗ, изменение премии отдельных инструментов. |
| Риск-сигналы | Ускоренный рост максимальной доходности, широкий диапазон min/max, высокая доходность при слабом спросе, отсутствие сопоставимой доходности по отдельным видам бумаг. |

## 3. Аналитика спроса и покрытия

| Элемент | Содержание |
|---|---|
| Цель | Измерить глубину спроса и устойчивость покрытия размещений. |
| Управленческие вопросы | Покрывает ли спрос предложение? Где спрос ослабевает? Какие периоды требуют изменения тактики размещений? |
| KPI | `bid_to_cover_ratio`, `demand_volume`, `demand_pressure_indicator`, `demand_satisfaction_ratio`. |
| Визуализации | Динамика bid-to-cover, распределение спроса по периодам, bubble chart спрос/доходность/объем. |
| Табличные outputs | Таблица совокупного спроса и предложения; агрегаты по `format`, `ofz_type`, `maturity_bucket`. |
| Потенциальные инсайты | Смена режима спроса, снижение конкурентности аукционов, усиление спроса на отдельные сроки. |
| Риск-сигналы | Bid-to-cover ниже 1, рост предложения без роста спроса, падение спроса на длинные выпуски. |

## 4. Аналитика спроса и предложения

| Элемент | Содержание |
|---|---|
| Цель | Сопоставить объем предложения Минфина и фактический спрос инвесторов. |
| Управленческие вопросы | Насколько предложение соответствует рыночной емкости? Как меняется дисбаланс спроса и предложения? |
| KPI | `total_demand`, `total_supply`, `demand_supply_ratio`, YoY-изменения спроса и предложения. |
| Визуализации | Dual-axis demand/supply, ratio line, stacked bars по формату и типу ОФЗ. |
| Табличные outputs | `demand_supply_<period_type>_<report_date>_retrospective_<N>.csv/.xlsx`. |
| Потенциальные инсайты | Периоды избыточного предложения, восстановление спроса после стрессовых периодов, изменение емкости рынка. |
| Риск-сигналы | Снижение `demand_supply_ratio`, нулевое или неполное предложение в расчетах, разрыв между планом и размещением. |

## 5. Аналитика структуры размещений

| Элемент | Содержание |
|---|---|
| Цель | Показать, какие типы бумаг, форматы и выпуски формируют размещение. |
| Управленческие вопросы | Смещается ли структура выпусков? Растет ли доля ДРПА? Какие типы ОФЗ доминируют? |
| KPI | Доля `format`, доля `ofz_type`, концентрация по выпускам, HHI размещения, `issue_placement_share_in_year`. |
| Визуализации | Stacked bar структуры, treemap выпусков, concentration chart, Sankey flow структуры выпусков. |
| Табличные outputs | Структурная таблица по `format`, `ofz_type`, `issue_code`, `report_period_label`. |
| Потенциальные инсайты | Рост концентрации заимствований, изменение роли ДРПА, переход от коротких выпусков к длинным. |
| Риск-сигналы | Чрезмерная концентрация в отдельных выпусках, резкое изменение формата размещений, снижение диверсификации. |

## 6. Аналитика сроков обращения

| Элемент | Содержание |
|---|---|
| Цель | Оценить сроковую структуру размещений и ее вклад в долговую устойчивость. |
| Управленческие вопросы | Удлиняется или укорачивается портфель новых размещений? Как меняется доля долгосрочных выпусков? |
| KPI | `placement_volume` по `maturity_bucket`, `placement_volume_share`, YoY-изменения объема и доли. |
| Визуализации | Stacked bar по срокам, maturity mix line, heatmap сроков, Sankey перераспределения сроковой структуры. |
| Табличные outputs | `placement_volume_by_maturity_<period_type>_<report_date>_retrospective_<N>.csv/.xlsx`. |
| Потенциальные инсайты | Переход размещений между кратко-, средне- и долгосрочным сегментами, изменение дюрационного профиля. |
| Риск-сигналы | Резкий рост краткосрочного сегмента, невозможность надежно определить срок, рост `requires_review`. |

## 7. Аналитика эффективности аукционов

| Элемент | Содержание |
|---|---|
| Цель | Оценить результативность аукционов с учетом размещения, спроса и доходности. |
| Управленческие вопросы | Какие аукционы дают лучший баланс объема и стоимости? Где размещение достигается ценой повышенной доходности? |
| KPI | `auction_efficiency_score`, `placement_to_offer_ratio`, `demand_to_offer_ratio`, `failed_or_no_deal`. |
| Визуализации | Efficiency quadrant, scatter efficiency/yield, ranking аукционов, distribution score. |
| Табличные outputs | Рейтинг аукционов по эффективности; сводка по периодам и типам ОФЗ. |
| Потенциальные инсайты | Выявление эффективных окон размещения, отделение сильного спроса от дорогого размещения. |
| Риск-сигналы | Низкая эффективность при высоком предложении, высокая доходность при слабом спросе, частые failed/no deal. |

## 8. Риск-аналитика

| Элемент | Содержание |
|---|---|
| Цель | Сформировать ранние сигналы риска для стоимости долга, спроса и структуры размещений. |
| Управленческие вопросы | Где накапливается риск? Какие метрики требуют управленческого внимания? |
| KPI | `yield_pressure_indicator`, `demand_pressure_indicator`, HHI, volatility, `requires_review`, `data_quality_flag`. |
| Визуализации | Risk quadrant, traffic-light dashboard, volatility chart, concentration heatmap. |
| Табличные outputs | Реестр риск-сигналов по периодам, форматам и срокам обращения. |
| Потенциальные инсайты | Совмещение роста доходности и снижения спроса, концентрация заимствований, ухудшение качества данных. |
| Риск-сигналы | High yield pressure, weak demand, рост концентрации, missing demand/supply/yield. |

## 9. Табличные аналитические отчеты

| Элемент | Содержание |
|---|---|
| Цель | Дать воспроизводимые таблицы для управленческого отчета и проверки выводов. |
| Управленческие вопросы | Какие числовые таблицы подтверждают выводы по доходности, спросу и срокам? |
| KPI | `yield_min`, `yield_weighted_avg`, `yield_max`, `total_demand`, `total_supply`, `demand_supply_ratio`, `placement_volume_share`. |
| Визуализации | Таблицы в dashboard, conditional formatting, sparklines по YoY. |
| Табличные outputs | `ofz_yield_by_type_*`, `demand_supply_*`, `placement_volume_by_maturity_*` в CSV/XLSX. |
| Потенциальные инсайты | Проверяемые числовые разложения для executive summary. |
| Риск-сигналы | `data_quality_flag` не `ok`, простое среднее вместо взвешенного, пустые периоды. |

## 10. Архитектура dashboard

| Элемент | Содержание |
|---|---|
| Цель | Организовать dashboard-ready outputs вокруг периодов, KPI, графиков и таблиц. |

## 11. Executive reporting

| Риск-сигналы | Неподтвержденные выводы, отсутствие таблиц, высокий уровень ограничений данных. |

## 12. Sankey flow перераспределения структуры выпусков

| Элемент | Содержание |
|---|---|
| Цель | Показать перераспределение структуры размещений между периодами по типам ОФЗ, форматам и срокам. |
| Управленческие вопросы | Из каких сегментов в какие смещается размещение? Какие структурные переходы объясняют изменение портфеля? |
| KPI | Объем размещения по `ofz_type`, `format`, `maturity_bucket`, изменение долей между периодами. |
| Визуализации | Sankey flow: период -> тип ОФЗ -> срок -> формат; альтернативно период -> maturity_bucket с YoY-потоками. |
| Табличные outputs | Матрица переходов долей и объемов по периодам; source-target table для Sankey. |
| Потенциальные инсайты | Перенос размещений между сроковыми сегментами, рост/снижение роли ДРПА, изменение типа выпусков. |
| Риск-сигналы | Резкие структурные потоки без подтверждения спросом, рост длинного сегмента при высокой доходности, пустые/неполные категории. |
